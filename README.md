## Структура таблиц базы данных

### 1. Студенты (`users.csv`) — 300 записей

| Название поля | Описание атрибута |
| :--- | :--- |
| `user_id` | Уникальный идентификатор пользователя |
| `Дата регистрации` | Дата создания аккаунта |
| `Город` | География пользователя |
| `Возраст` | Возраст студента |
| `Email` | Электронная почта |
| `Телефон` | Контактный номер |
| `Канал` | Текстовое название источника трафика |
| `chanel_id` | Идентификатор маркетингового канала |

### 2. Список курсов (`courses.csv`) — 30 записей

| Название поля | Описание атрибута |
| :--- | :--- |
| `course_id` | Уникальный идентификатор курса |
| `Название курса` | Наименование образовательного продукта |
| `Цена` | Розничная стоимость для клиента |
| `Себестоимость` | Внутренние затраты на производство и поддержку |
| `Налог` | Ставка налогообложения (0.06) |
| `Категория` | Направление обучения (IT / Маркетинг / Дизайн) |

### 3. Заказы (`orders.csv`) — 600 записей

| Название поля | Описание атрибута |
| :--- | :--- |
| `order_id` | Уникальный номер транзакции |
| `user_id` | Идентификатор студента |
| `course_id` | Идентификатор приобретенного курса |
| `Дата заказа` | Дата оформления транзакции |
| `Статус` | Текущее состояние заказа (Оплачен / Отменен / Возврат) |
| `Промокод` | Примененный маркетинговый код |
| `Скидка` | Размер дисконта |
| `Цена` | Базовая цена продукта |
| `Финальная цена` | Итоговая стоимость с учетом скидки |
| `Себестоимость` | Расходная часть продукта |
| `Налог` | Сумма налоговых отчислений |
| `Канал` | Маркетинговый источник продажи |

### 4. Маркетинговые расходы (`marketing_costs.csv`)

| Название поля | Описание атрибута |
| :--- | :--- |
| `Дата` | День фиксации затрат |
| `Канал` | Источник привлечения трафика |
| `Расходы` | Сумма затрат на рекламу |
| `Показы` | Количество демонстраций объявлений |
| `Клики` | Количество переходов по рекламе |
| `chanel_id` | Идентификатор рекламного канала |

### 5. Активность студентов (`activity.csv`)

| Название поля | Описание атрибута |
| :--- | :--- |
| `user_id` | Идентификатор студента |
| `Дата посещения` | День активности в личном кабинете |
| `Завершенные уроки` | Количество пройденных занятий |
| `Баллы` | Оценка успеваемости за тесты |
